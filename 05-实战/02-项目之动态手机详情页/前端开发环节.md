## 居中问题小结

1. 文字垂直居中

   ```css
   .item {
     height: 38px;
     line-height: 38px;
   }
   ```

2. 块元素垂直居中于父元素

   ```css
   .container {
     position: relative;
   }
   .item {
     height: 38px;
     position: absolute;
     top: 50%;
     margin-top: -19px; /* height的一半 */
   }
   ```

3. 块元素水平居中

   ```css
   // 法一
   .container {
     width: 1200px; /* 宽度一定得有，否则居中失效 */
     margin: 0 auto;
   }

   // 法二：负边距
   .container {
     width: 100px; /* 宽度一定得有，否则居中失效 */
     left: 50%;
     margin-left: -50px; /* width的一半，且为负 */
   }
   ```
