# 选择器

- 全局选择器：\* { }
- 标签选择器：标签名 { }
- id 选择器：#id { }
- 类选择器：.类名 { }
- 属性选择器：[属性] { }
- 伪类选择器
  ```
  :link 超链接点击之前
  :visited 超链接点击之后
  :hover 鼠标悬停
  :active 鼠标点击不松手
  :focus 获取焦点时
  :target 当在文档内部使用 ID 作为链接的时候，用户点击了这个 ID 链接，被链接的元素上会应用 target 设置好的样式
  :disabled input禁用时的样式
  ......
  ```
- 伪元素选择器
  ```
  ::first-line 首行
  ::first-letter 首字
  ::before 之前
  ::after 之后
  ......
  ```
