# 盒阴影

## box-shadow：可以设置一个或多个下拉阴影的框

```css
.box {
  box-shadow: h-shadow v-shadow blur spread color inset;
}
```

- h-shadow：必需，表示水平阴影的位置，**正值阴影向右，负值向左**
- v-shadow：必需，表示垂直阴影的位置，**正值阴影向下，负值向上**
- blur：可选，代表模糊距离，**不能为负**
- spread：可选，阴影的尺寸扩展，**往 4 个方向扩展**
- color：可选，阴影的颜色
- inset：可选，将外部阴影转换成内部阴影
