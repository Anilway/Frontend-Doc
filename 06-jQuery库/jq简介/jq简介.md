# jq 简介

1. 作用

   - HTML 元素获取
   - HTML 元素操作
   - CSS 操作
   - 事件
   - js 特效与动画
   - DOM 遍历与修改
   - AJAX
   - Utilities

2. 目的

   - 写得更少，做得更多

3. 预备知识

   - HTML
   - CSS
   - JavaScript：任何情况 JS 都是最重要的，不要忘本

4. 优势

   - 轻量
   - 强大的选择器
   - 出色的 DOM 操作及封装
   - 可靠的事件
   - 完善的 AJAX
   - 不污染顶级变量
   - 出色的浏览器兼容性
   - 支持链式操作
   - 隐式迭代
   - 结构与行为分离
   - 丰富的插件
