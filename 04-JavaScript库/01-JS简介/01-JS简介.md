### JavaScript 分为三个部分：

- ECMAScript：JavaScript 的语法标准

- DOM：文档对象模型，提供**访问和操作网页内容的 API**，分为 3 个级别

- BOM：浏览器对象模型，提供**与浏览器交互的 API**

### JS 特点：

1. 弱变量类型
2. 解析型语言：事先不需要被编译为机器码再执行（编译型），而是边解析边执行
3. 基于原型
4. 单线程

### 规范

- 务必在严格模式 strict 下写 JS 代码
  - 开头写上`"use strict";`
