# 功能类元素

- h：标题 1~6
- p：段落
- a：链接

  ```
  真链接、假链接、锚链接
  target属性：
      _self 在当前页打开（默认）
      _blank 新页面打开
      _parent 在父级页面打开（链接在<iframe>内部的情况下）
      _parent 在顶级级页面打开（链接在嵌套的<iframe>内部的情况下）
  ```

- img：图片

  ```
  JPEG 高压缩比，通常后缀为.jpg或.jpeg
  GIF 可做动图，通常后缀为.gif
  PNG 带有不透明度通道，通常后缀为.png
  SVG 矢量图形，通常后缀为.svg
  ```

- form：表单
- button：按钮，一般配合 JS 使用
- input：用户输入

  ```
  type属性:
      text 文本框
      password 用于输入密码
      radio 单选，限定了选择范围的同时只能选中一项,name需一致
      checkbox 多选框，限定了选择范围的同时还可以多选,name需一致
      submit 提交按钮
      reset 重置按钮
      select/option 下拉选择框
      textarea：多行文字输入框, 通常用于输入多行文字，如文章，备注，评论
      color 通过调用浏览器的色盘选择颜色
      date 用于选择日期
      number 用于输入数字
      ......
  checked 可以使选框默认选中
  placeholder 底纹
  autofocus 有时候一张页面打开后我们希望能直接"将光标放到主输入框中"，比如说我们在做一个搜索页
  ```

- table：表格

  ```
  <table>
      <caption>表站花名册</caption>
      <thead>
          <tr>
              <th>姓名</th>
              <th>年龄</th>
          </tr>
      </thead>
      <tbody>
          <tr>
              <td>王花花</td>
              <td>18</td>
          </tr>
          <tr>
              <td>李拴蛋</td>
              <td>19</td>
          </tr>
          <tr>
              <td>赵可爽</td>
              <td>20</td>
          </tr>
      </tbody>
  </table>
  ```

- 列表

  ```
  <ul>  定义无序列表
  <ol>  定义有序列表
  <li>  定义列表项
  <dl>  自定义列表
  <dt>  自定义列表项
  <dd>  定义自定列表项的描述
  ```

- iframe：内嵌框架
- video/audio：视频/音频
